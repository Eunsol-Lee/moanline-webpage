# 모앤라인성형외과 홈페이지

헤어라인교정술, 모발이식, 탈모치료 전문 — 모앤라인성형외과 공식 웹사이트

## 기술 스택

- HTML5
- CSS3 (CSS Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Google Fonts (Noto Sans KR)
- Font Awesome 6

## 실행 방법

정적 웹사이트이므로 별도 빌드 없이 실행 가능합니다.

### 방법 1: 파일 직접 열기
```bash
open index.html
```

### 방법 2: 로컬 서버 실행
```bash
# Python
python3 -m http.server 8000

# Node.js (npx)
npx serve .
```
브라우저에서 `http://localhost:8000` 접속

## 프로젝트 구조

```
moanline-webpage/
├── index.html          # 메인 페이지 (원페이지 스크롤)
├── css/
│   └── style.css       # 전역 스타일 (반응형 포함)
├── js/
│   └── main.js         # 인터랙션 (네비게이션, 애니메이션, 폼)
├── assets/
│   └── images/         # 이미지 에셋
└── README.md
```

## 주요 기능

- 히어로 섹션 (슬로건, CTA)
- 진료 과목 소개 (헤어라인교정술, 모발이식, 탈모치료)
- 대표원장 소개
- Before/After 갤러리
- 오시는 길 (주소, 운영시간)
- 온라인 상담 예약 폼
- 반응형 디자인 (모바일/태블릿/데스크탑)
- 고정 네비게이션 + 모바일 햄버거 메뉴
